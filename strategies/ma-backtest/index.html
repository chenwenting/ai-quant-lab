双均线策略

同时计算两条不同周期的简单移动平均线（SMA），通过交叉关系判断趋势方向。

短期均线捕捉动能变化，长期均线确定趋势方向。趋势策略，不预测只跟随。

▲ 金叉 — 买入信号

短期均线从下方上穿长期均线。短期动能超过长期趋势 → 买入建仓。

▼ 死叉 — 卖出信号

短期均线从上方下穿长期均线。动能衰减 → 清仓离场。

连续同向信号被忽略，避免频繁交易侵蚀利润。可调仓位比例（全仓/半仓/1/3仓）控制风险。

指标概念速查
累计回报
策略总收益率：(最终资产 - 初始资金) / 初始资金。正值为盈利，负值为亏损。不分时间长短，跨周期对比见年化收益。
年化收益
折合"平均每年赚多少"： (1 + R)^(252 / T) - 1。R = 累计回报，T = 交易日数。
最大回撤 MDD
MDD = max( (Peak − Trough) / Peak )。Peak 为历史最高净值，Trough 为该峰值之后的谷底净值。遍历每日，取全程最大回撤比例。
夏普比率
风险调整后收益：(年化收益 - 2.5%) / 年化波动率。< 0 跑输无风险，> 1 良好，> 2 优秀。
胜率
盈利交易次数 / 总交易次数。需结合盈亏比评估，高胜率不等于高收益。
总交易次数
完整买卖配对数量。同向信号被忽略。交易频繁时手续费侵蚀利润，可切换万分之一费率对比。
盈亏比
总盈利金额 / 总亏损金额。> 1 策略整体盈利，< 1 策略亏损，> 2 为优秀。回撤大时需结合MDD判断。
